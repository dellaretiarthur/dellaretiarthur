#  About Me:
Computer Science student with a security-first mindset, specializing in both defensive (Blue Team) and offensive (Red Team) security domains. My work bridges the gap between infrastructure and application security to detect, mitigate, and prevent threats.

Originally trained in Architecture and Urban Design, I bring a unique perspective to complex systems: deconstructing risk, identifying failure points, and designing layered defenses. This background translates directly into security analysis, incident response, and penetration testing.

Ultimately, security is a mindset, not a checklist. I'm here to build, break, and secure systems in a practical, measurable, and resilient way.
